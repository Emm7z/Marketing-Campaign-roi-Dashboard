# Marketing Campaign & ROI Optimization Dashboard

## Project Overview
This project focuses on improving visibility into marketing campaign performance across Ads, Email, and Social Media channels. The goal was to provide actionable insights that optimize ROI and guide future campaign decisions.

## Challenge
Limited visibility into campaign performance made it difficult for the marketing team to identify top-performing campaigns and allocate resources effectively.

## Solution
Built an **interactive Tableau dashboard** displaying key metrics such as campaign conversion rates, ROI, and A/B testing results. The dashboard allows stakeholders to analyze performance by channel, campaign type, and date range.

## Results
- Identified the top-performing campaign ("Holiday Promo") with an ROI of 2.18  
- Determined Ads as the most efficient channel with a 22.5% conversion rate  
- Provided actionable insights to optimize marketing investments and future campaigns
